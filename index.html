Người Đam mê Machine Learning & Deep Learning

Tôi đam mê nghiên cứu và áp dụng các mô hình Machine Learning & Deep Learning vào thực tế. Định hướng của tôi là trở thành một nhà nghiên cứu AI (AI Researcher). Phương châm của tôi là tự học hỏi, mày mò nghiên cứu lý thuyết qua các khoá học chuyên sâu và chuyển hóa nó thành dòng code với Python và C++.

Mục tiêu nghề nghiệp

Mục tiêu lớn nhất của tôi là theo đuổi con đường nghiên cứu chuyên sâu về Trí tuệ Nhân tạo, đặc biệt trong lĩnh vực Machine Learning và Deep Learning. Dự án Nổi bật

CS231n - Deep Learning for CV

Hoàn thành và nghiên cứu các bài tập từ khóa CS231n của Stanford. Thực hành code các mạng lưới Nơ-ron, K-nn, Softmax, Transformer, và Convolutional Networks từ con số 0 với Python.

Khám phá

Machine Learning Practice

Triển khai các mô hình Machine learning cốt lõi và củng cố nền tảng Toán học. Nội dung bao gồm việc xây dựng Classification (Phân lớp), Linear Regression (Hồi quy tuyến tính) và thuật toán Unsupervised (Thuật toán không giám sát).
